REVISIÓN DE FIGURAS

Cómo se usa: cada tarjeta es un grupo: si tiene más de una imagen, se fusionarán en UN mosaico y Sonnet lo describirá con UNA sola llamada. Arrastra una imagen a otro grupo para moverla (o suéltala sobre otra imagen para ordenar), usa ＋ Grupo nuevo para separar una figura, edita el pie directamente en el cuadro de texto, ✂ Recortar abre el editor de recorte, 🗑 manda la imagen a descartadas (logos, decoración), ⇪ reemplazar sustituye una imagen mal extraída por una captura tuya (p.ej. de la herramienta Recortes de Windows sobre el PDF) y ⇪ Subir imagen agrega una captura nueva al grupo. Cuando todo esté bien: ✔ APROBAR PLAN. Nada se envía a las APIs hasta aprobar.

🗑 DESCARTADAS (no se describen ni se indexan) — arrastra aquí para descartar, o de vuelta a un grupo para recuperar
